# Кабинет скульптора
**0.07c Кабинет скульптора (каталог с индивидуальными разделами)**
React-приложение с CSS-модулями: заставка, главное меню, каталог инструментов по категориям и подгруппам, страницы предметов с описанием и галереей.

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`.

## Данные

Данные каталога лежат в **`public/data/`** в формате JSON:

- **`public/data/catalog.json`** — структура: категории → подгруппы → предметы (название, описание, особенности, назначение, изображения, типы и материалы для фильтров).

Изображения для каталога можно положить в **`public/data/images/`** (и в подпапки, например `items/`), и указывать пути в JSON от корня сайта (например `/data/images/items/wl-1.jpg`).

## Фон страниц

Папка **`src/assets/`** предназначена для фоновых изображений страниц. После добавления файлов их можно подключать в компонентах и задавать в CSS как `background-image` (например через `import` или путь из `public`).

## Маршруты

- `/` — заставка (через ~2.5 с переход на меню)
- `/menu` — главное меню («Перейти в каталог», «Участвовать в викторине», кнопка «Назад»)
- `/catalog` — каталог (первая категория), переключение по 4 категориям в боковой панели
- `/catalog/:categoryId` — каталог с выбранной категорией
- `/catalog/:categoryId/:subgroupId` — каталог подгруппы (до 10 предметов, фильтры по видам/материалам, поиск по названию)
- `/catalog/:categoryId/:subgroupId/item/:itemId` — страница предмета (хлебные крошки, текст с пагинацией, галерея фото, полноэкранный режим)
